# Media Sentiment and Market Volatility

This repository contains the working paper and supporting materials for my research on the influence of media sentiment on financial market volatility.

## Current Contents

- Drafts of the Introduction, Literature Review, and Methodology sections.
- R script for the baseline log-return EGARCH model (`Base_logreturn_Egarch.R`).

## Upcoming

- Sentiment analysis code will be added soon, incorporating media sentiment metrics into the volatility model.

## Status

Work in progress. Manuscript expected to be completed in 2025.
